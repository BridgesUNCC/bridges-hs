Description

In this project, students read elevation data into a 2D array, compute the path of least elevation change using the greedy algorithm, and visualize the path using the ColorGrid.

While the primary focus of this project is 2D arrays, it covers a number of other important topics as well. Students generally find it very fun and engaging because it applies to the real world. The project can be adapted to fit the ability of students and has a lot of potential for variation.

This project is adapted from a Nifty assignment created by Baker Franke in 2016. Learn more

Greedy algorithm

A greedy algorithm is an algorithm that makes locally optimal decisions to generate a global optimum. In this project, the “optimal” is defined as the decision that involves the least change in elevation. Consider the following figure:

In each case, the green box indicates the “optimal” decision based on the specified criteria: least elevation change. Notice the last two cases:

  1. Case 3: When the forward elevation change is equivalent to the up and/or down elevation changes, the forward choice is preferred.
  2. Case 4: When the up and down elevation changes are equivalent, flip a coin to determine which way to go.

The central premise of the project is to use the greedy algorithm to make this decision repeatedly, generating a globally optimal path.

Intended learning outcomes

Through this project, students develop the ability to...

  • Apply: Implement file path operations for accessing data files.
  • Understand: Identify the purpose of String[] args in the context of the main method.
  • Understand: Analyze and explain code that reads data files.
  • Understand: Analyze and explain code that converts elevation data to a 2-dimensional array and creates a grayscale image.
  • Remember: Define the greedy algorithm.
  • Apply: Develop a greedy algorithm as a heuristic for finding a “least elevation change” path given elevation data.
  • Create: Create a visual representation of the path using the BRIDGES ColorGrid.
  • Evaluate: Critique the greedy algorithm as a means of achieving the optimal path.

Reference for approach: Bloom’s Taxonomy of Educational Objectives - Cognitive Domain
